# Dragon Indexes — cloud-native sequence-search indexes for bacterial genomics

> **DRAFT — for internal review before publication.** Placeholders are marked `«…»` / `[TBD]`. Verify every value before going public.

**Dragon** builds compact, shardable indexes that let you align a query sequence against
millions of bacterial genomes (and reference gene catalogues) in seconds. This release
publishes the prebuilt indexes so anyone can search them without rebuilding from scratch.

Indexes are distributed as **Zarr** (cloud-native, chunked) so you can either:
- **Stream** directly from S3 (no full download — Dragon/Zarr fetch only the chunks they need), or
- **Download** a single `.tar.zst` per index for fully local use.

## Access

- **Bucket:** `s3://dragon-zarr/` &nbsp;·&nbsp; **Region:** `eu-west-2`
- **HTTPS base:** `https://dragon-zarr.s3.eu-west-2.amazonaws.com/`

> On acceptance into the **AWS Open Data Sponsorship Program**, storage + egress become free
> to users (no `--request-payer` needed) and the bucket is registered in the
> [Registry of Open Data on AWS](https://registry.opendata.aws/). Until then, access may be
> requester-pays. See [`aws-open-data/`](aws-open-data/).

### List an index
```bash
aws s3 ls s3://dragon-zarr/atb/ --region eu-west-2 --no-sign-request
```

### Download one index (single archive)
```bash
# «confirmar: caso publiquemos tarballs por índice»
aws s3 cp s3://dragon-zarr/_archives/saureus.tar.zst . --region eu-west-2 --no-sign-request
tar --use-compress-program=unzstd -xf saureus.tar.zst
```

### Stream directly from S3 (no download)
```bash
# «confirmar a sintaxe exata do Dragon para ler índice/zarr em S3»
dragon search \
  -i s3://dragon-zarr/atb/shard_0000.zarr \
  --shard s3://dragon-zarr/atb/shard_0001.zarr \
  -q my_query.fa -o hits.paf \
  --threads 8
```

## Quickstart (local)

```bash
# 1. install Dragon  «link de instalação / cargo install / release binary»
# 2. fetch an index (or stream from S3 as above)
# 3. search
dragon search -i ./saureus_index -q amr_genes.fa -o hits.paf --preset amr --threads 8
# 4. results are PAF; columns 1=query 6=target 10=matches 11=aln_len
```
